Applet:

Applet is a spacial type of program that is embbed in the web-page to generate the dynamic content.

It runs on browser and work as client side.

No main() method / always start from init() Method.

Secqured

It can be executed by browser running under many platforms including linux, macos, windows, bsds etc.

It works as client side so less respoonse time.

Plugin is required at client browser to execute applet (Drawback).

Life Cycle of applet:

  1. init():

    public void init();

    init() called as initialization because there is no main method unlike our normal java program. Every applet start with init() method.

    called only once a program

  2. start():

    public void start();

    After init() method start() method is invoked or Executed when the browser is maximized.

    Contains actual code that has to be executed.

  3. paint():

    public void paint(Graphics g);

    paint() method is used to display the content on the applet.

    We can create the objects or components to the applet or we can directly write a message on the applet.

    It will take Graphics class as a parameter.

  4. stop():

    public void stop();

    Temperory stop or pushed the applet. It is executed when the browser is minimized.

  5. destroy():

    public void destroy();

    It's used to completely close the applet. It's executed when the applet is closed.

    It also execute only once.

Param Keyword/Tag:

Also known as parameter passing technique in applet.

We can pass parameter in applet through html using param Keyword.

Html tag: param two attributs: name and value (simply pass the string by this).

Note:We are passing string. If you give integer value it also convert that into a string and then pass the data.

For geting a string, getParameter() method is used.

Param Tag Attributes:

Attributes name Value Description
Code URL It specifies the URL of java.class file

Height Pixels Specifies the height of applet
hspace Pixels Specifies the horizontal space around applet.
vspace Pixels Specifies the vartical space around applet.
